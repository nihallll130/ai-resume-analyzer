## About
AI-powered resume analyzer that gives ATS compatibility 
scores, keyword gap analysis, and AI-improved bullet points.

## Features
- ATS Score (0-100)
- Missing keyword suggestions
- AI bullet point rewriter

## Built With
Python, Streamlit, spaCy, scikit-learn, OpenAI API, PyMuPDF
